叔叔节点红色

case 1 - 叔叔节点红色

关注节点 node，其叔叔节点是红色
此情况下必然存在祖父节点 gParent，且祖父节点一定是黑色。

操作步骤：

  1. 将关注节点 node 的父节点 parent，叔叔节点 uncle 都染黑；
  2. 将关注节点 node 的祖父节点 gParent 染红；
  3. 将关注节点 node 指向祖父节点 gParent；
  4. gParent 染红了，如与其父形成连续的红色节点，此时继续走 case1；
  5. 没有跳到 case 2 或 case 3。
