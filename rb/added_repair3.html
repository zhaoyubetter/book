叔叔黑且 node 为父的左孩子

case 3 - 叔叔黑且关注节点为父的左孩子

关注节点 node，其叔叔节点黑色，并且关注节点 node 是其父的左孩子；

操作步骤：

  1. 将关注节点的父染黑，并将祖父节点染红
  2. 围绕关注节点node的祖父节点右旋；
  3. 根节点染黑；
  4. 调整结束。
